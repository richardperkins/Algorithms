# Dictionary
**Dictionaries** use named index pointers to assign and retrieve data. 

| Dictionary operation | Unsorted Array | Sorted Array |
|----------------------|----------------|--------------|
| $$Search(L, k)$$ | $$O(n)$$ | $$O(log n)$$ |
| $$Insert(L, x)$$ | $$O(1)$$ | $$O(n)$$ |
| $$Delete(L, x)$$ | $$O(1)$$ | $$O(n)$$ |
| $$Successor(L, x)$$ | $$O(n)$$ | $$O(1)$$ |
| $$Predecessor(L, x)$$ | $$O(n)$$ | $$O(1)$$ |
| $$Minimum(L)$$ | $$O(n)$$ | $$O(1)$$ |
| $$Maximum(L)$$ | $$O(n)$$ | $$O(1)$$ |
